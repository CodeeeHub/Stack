بررسی داده ساختار Stack در جاوا اسکریپت

پشته یک داده ساختار مهم در الگوریتمه که بر مبنای لیفو عمل می کنه یعنی آخرین عنصری که وارد میشه اولین عنصریه که خارج میشه.

داده ساختار مهمی که خیلی پر کاربرد و پای ثابت مصاحبه ها هستش.

پشته یک داده ساختار انتزاعی یعنی چی ?

یعنی این که پشته به صورت فیزیکی داخل حافظه پیاده سازی نمیشه و ما میتونیم با داده ساختار های دیگه مثل آرایه و لیست پیوندی و ... پشته رو پیاده سازی کنیم.

این داده ساختار این طوری عمل می کنه که :
  • در پشته برای اضافه کردن یا برداشتن هر عنصر ما باید جایگاه بالاترین عنصر رو بدونیم , بیشتر اوقات این مقدار در متغیری به اسم Top ذخیره میشه.
  • در ابتدا که پشته خالیه و هیچ عنصری داخل آن وجود نداره مقدار TOP برابر 1- قرار میدییم.
  • پشته دو متد اصلی به نام های PUSH که برای اضافه کردن مقدار به پشته و POP که برای برداشتن مقدار از پشته است.
  • مقدار TOP در PUSH یکی اضافه میشه ولی در POP یکی از آن کم می کنیم.
داخل عکس زیر نحوه کار پشته رو می بینید:

هر داده ساختاری در یک موضوع به خصوص قوی و به درد بخور است و در یک موضوع ضعیف و نقطه ضعف داره که نقاط ضعف و قوت پشته رو با هم بررسی کنیم :

از نقاط ضعف اون میتونیم اشاره کنیم به :

  • حداکثر تعداد عناصری که یک پشته میتونه نگه داری بکنه محدوده و ما در پشته مشکل اندازه محدود رو داریم .
  • ما در پشته فقط میتونیم به عنصر اخر دسترسی پیدا کنیم و عملا مفهوم پیمایش کردن در پشته جایگاهی نداره.
  • در برخی پیاده سازی ها ممکنه باغث بشه که در حافظه مشکلاتی ایجاد بشه.

از نقاط قوت اون میتونیم اشاره کنیم به :

  • همان طور که قبلا گفته شد الگوریتم پشته یک نوع از الگوریتم های انتزاعی است پس میتوان با آرایه , لیست پیوندی و ... برای پیاده سازی این الگوریتم استفاده کرد.
  • این الگوریتم در زبان های مختلف پیاده سازی ساده ای داره.
  • این الگوریتم نسبت به بقیه الگوریتم ها مانند صف سرعت بیشتری داره.

نکات مهمی که در باره پشته باید بدونیم اینه که :

  • زمان اجرای هر کدان از عملیات PUSH , POP در پشته ثابت است و از نوع(1) O است.
  • پشته بیشتر در فراخوانی توابع بازگتشی کاربرد داره , هنگامی که یک تابع فراخوانی میشه اطلاعات و ترتیب اجرا توابع در پشته ذخیره میشه.
  • در بسیاری از الگوریتم‌ها مانند الگوریتم DFS (جستجوی عمق اول) و حل مسائل گرافی، پشته به عنوان یک ابزار مؤثر برای ذخیره و بازگشت به اطلاعات استفاده می‌شود.
یک نمونه مثال از پشته رو می بینید که در جاوا اسکریپت نوشته شده : 
          class Stack {
               constructor() {
                 this.items = [];
               }
             
               push(element) {
                 this.items.push(element);
               }
             
               pop() {
                 if (this.items.length === 0) {
                   return "Underflow";
                 }
                 return this.items.pop();
               }
             
               peek() {
                 return this.items[this.items.length - 1];
               }
             
               isEmpty() {
                 return this.items.length === 0;
               }
             
               size() {
                 return this.items.length;
               }
             
               display() {
                 console.log(this.items);
               }
             }
             
             // Example usage:
             const stack = new Stack();
             stack.push(1);
             stack.push(2);
             stack.push(3);
             stack.display(); // Output: [1, 2, 3]
             
             stack.pop();
             stack.display(); // Output: [1, 2]
             
             console.log("Peek:", stack.peek()); // Output: Peek: 2
             console.log("Size:", stack.size()); // Output: Size: 2
             console.log("Is Empty:", stack.isEmpty()); // Output: Is Empty: false
